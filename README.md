# ReserVamos
ReserVamos is a program made in the JAVA IDE that let's the user to make reservations of dinner tables whit great variety of restaurants, trough a downloadable app prototype so that the idea clould be  developed further in the future projects, such as a web app or mobile app (non applied in this run).
